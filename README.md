# Pocket

Pocket is a modern event planning and profitability platform designed for retreat organizers, travel operators, event creators, and experience-based businesses.

The platform helps teams replace complex spreadsheets with a structured workflow for planning events, managing costs, modeling scenarios, and understanding profitability.

## Features

### Dashboard

* Business overview
* Revenue tracking
* Expense tracking
* Profitability metrics
* Quick insights

### Event Management

* Create events
* Manage event details
* Track event performance
* Centralized planning workspace

### Package Builder

* Create event packages
* Define pricing structures
* Manage participant capacity
* Revenue forecasting

### Expense Management

* Add and categorize expenses
* Track operational costs
* Monitor spending in real time

### Scenario Planning

* Compare pricing strategies
* Model attendance variations
* Evaluate profitability outcomes

### Snapshots

* Save planning states
* Compare historical scenarios
* Analyze decision impact

### Reporting

* Event profitability reports
* Revenue summaries
* Expense analysis
* Operational insights

## Getting Started

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Start production server:

```bash
npm start
```
